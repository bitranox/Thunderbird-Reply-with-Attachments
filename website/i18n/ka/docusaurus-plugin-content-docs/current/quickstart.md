## სწრაფი დაწყება

:::important Thunderbird-ის მინიმალური ვერსია
ეს გაფართოება მუშაობს Thunderbird-ის **128 ESR ან უფრო ახალ** ვერსიებზე. ძველი ვერსიები არ არის მხარდაჭერილი.
:::

:::note ტელემეტრიის გარეშე; ფონური ქსელი — არა
გაფართოება **არ** აგროვებს ანალიტიკას/ტელემეტრიას და **არ** აგზავნის ფონურ ქსელურ მოთხოვნებს. ქსელთან წვდომა ხდება მხოლოდ იმ დროს, როდესაც არის გარე ბმულებზე დაწკაპება (Docs, GitHub, Donate).
:::

---

### დაყენება

1. დააყენეთ გაფართოება Thunderbird Add‑ons-დან.
2. სურვილისამებრ: ჩართეთ დადასტურება (Options → “Ask before adding attachments”).
3. სურვილისამებრ: დატოვეთ შავი სიის გაფრთხილება ჩართული (ნაგულისხმევად): “Warn if attachments are excluded by blacklist”.
4. სურვილისამებრ: დაამატეთ შავი სიის შაბლონები (თითო ხაზზე ერთი), მაგ.:

```
*intern*
*secret*
*passwor*  # matches both “password” and “Passwort” families
```

შენიშვნა: ზემოთ არსებული “# …” ამ დოკუმენტაციაში კომენტარია; Options-ში ჩასმისას შაბლონებში კომენტარები არ ჩასვათ. თითო ხაზზე შეიყვანეთ მხოლოდ ერთი შაბლონი.

ახლა უპასუხეთ მიმაგრებებით მქონე წერილს — ორიგინალები დაემატება ავტომატურად ან მოკლე დადასტურების შემდეგ. თუ რომელიმე ფაილი თქვენი შავი სიის გამო გამოირიცხება, დაინახავთ მოკლე გაფრთხილებას მათი ჩამონათვალით.

---

### შემოწმება {#verify}

- უპასუხეთ წერილს 1–2 მიმაგრებით და დარწმუნდით, რომ ორიგინალები დაემატა წერის ფანჯარაში.
- ქცევის მოსარგებად იხილეთ [კონფიგურაცია](configuration) (დადასტურების გადამრთველი, ნაგულისხმევი პასუხი, შავი სიის შაბლონები).

---

### შავი სიის გაფრთხილების შემოწმება {#verify-blacklist-warning}

- უპასუხეთ წერილს, სადაც არის ისეთი ფაილი, როგორიცაა “secret.txt”.
- როცა ჩართულია “Warn if attachments are excluded by blacklist”, მცირე დიალოგში გამოჩნდება გამორიცხული ფაილების სია და შესაბამისი შაბლონი.

თუ გაფრთხილებას ვერ ხედავთ, დარწმუნდით, რომ შაბლონი ზუსტად ემთხვევა ფაილის სახელს (მხოლოდ ფაილის სახელი, რეგისტრის მიმართ უგრძნობი). იხილეთ კონფიგურაცია → შავი სია.

---

### კლავიატურის შენიშვნა {#keyboard-note}

- დადასტურების დიალოგი მხარს უჭერს Y/J-ს Yes-ისთვის და N/Esc-ს No-სთვის. ზოგ არალათინურ კლავიატურაზე ასოები შეიძლება განსხვავდებოდეს; Enter დაადასტურებს ფოკუსირებულ ღილაკს.
