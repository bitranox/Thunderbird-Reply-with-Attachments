## ተስማሚነት {#compatibility}

:::important ዝቅተኛ የThunderbird ስሪት
ይህ አድ‑ኦን የThunderbird **128 ESR ወይም ከዚያ በላይ**ን ይደግፋል። ነባር ስሪቶች አይደገፉም።
:::

- የThunderbird 128 ESR ወይም ከዚያ በላይ ይደገፋል (128.x+).
- ነባር የThunderbird ስሪቶች አይደገፉም።
- በWindows፣ macOS፣ እና Linux ላይ ተሞክሮ ተደርጓል።

:::warning Manifest V2 አይደገፍም
Manifest V2 አይደገፍም። አድ‑ኦኑ በThunderbird 128 ESR+ ላይ ለ**Manifest V3 (MV3)** ተመርኮዝ ነው።
:::

## ችግኝ መፍትሄ {#troubleshooting}

- አድ‑ኦኑ እንደተጠበቀው ካልሰራ፣ ተስማሚ የThunderbird ስሪትን (128 ESR ወይም ከዚያ በኋላ) እንደምትጠቀሙ ያረጋግጡ።
- ከአድ‑ኦኑ ጋር የተያያዙ ጉዳዮችን ለማግኘት የThunderbird ስህተት ኮንሶልን (**Tools > Developer Tools > Error Console**) ይመልከቱ።
- የተቀመጡ ቅንብሮች በትክክል እንዳይተገበሩ ከታዩ፣ Thunderbirdን ዳግም ያስጀምሩ እና እንደገና ይሞክሩ። (Thunderbird ክፍለ-ጊዜዎች መካከል ሁኔታን ሊካሽ ይችላል፤ እንደገና መጀመር አዲስ ቅንብሮች እንዲጫኑ ያረጋግጣል።)

## የታወቁ ግጭቶች {#known-conflicts}

- በአሁኑ ጊዜ የታወቀ ምንም ግጭት የለም። ተደጋጋሚ አባሪ ፋይሎች ወይም ከሌሎች አድ‑ኦኖች ጋር የሚከሰቱ ውድቀቶችን ካዩ፣ እባክዎ ለመድጋገም ደረጃዎቹን ጨምሮ ያሳውቁ።

---

## ቤታ/ናይትሊ {#beta-nightly}

- የThunderbird ቤታ እና Daily (Nightly) ሕንፃዎች በአብዛኛው ይደገፋሉ፣ ግን ከላይ የሚመጡ ለውጦች አንዳንድ ጊዜ የአድ‑ኦኑን ባህሪ ሊፈርሙ ይችላሉ። ችግኝ ካጋጠማዎት እባክዎ ያሳውቁ እና ትክክለኛውን የThunderbird ስሪት (ለምሳሌ፣ “Daily 131.0a1 2025‑09‑10”) ያካትቱ።
- በቤታ/ናይትሊ ላይ ሬግሬሽን አጋጥሞዎታል? እባክዎ ለመድጋገም ደረጃዎቹን ጨምሮ ያሳውቁ — ይመልከቱ [ድጋፍ](support) — እና ትክክለኛውን የThunderbird ህንፃ መለያ ሐረግ ያካትቱ።
