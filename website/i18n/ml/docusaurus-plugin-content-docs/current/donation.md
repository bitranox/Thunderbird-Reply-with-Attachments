## സംഭാവന ദൃശ്യമാനം (90‑ദിവസ സ്നൂസ്)

സംഭാവന നൽകിയതിന് ശേഷം കുറച്ചുകാലത്തേക്ക് സംഭാവന പ്രോംപ്റ്റുകൾ മറയ്ക്കുന്ന ഒരു സൗകര്യ സവിശേഷത ആഡ്‑ഓണിൽ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്.

- ഇത് എവിടെ കണ്ടെത്താം
  - Options → Support വിഭാഗം: നിങ്ങൾക്ക് “I donated” ബട്ടണും ചെറിയൊരു സൂചന പ്രദേശവും കാണാം.
  - Send‑confirmation ഡയലോഗിലും Donate ബട്ടൺ കാണിക്കും; സ്നൂസ് സജീവമായിരിക്കുമ്പോൾ അത് സ്വയം മറയും.

- ഇത് എങ്ങനെ പ്രവർത്തിക്കുന്നു
  - “I donated” ക്ലിക്ക് ചെയ്താൽ സംഭാവന ബട്ടണുകളും ബന്ധപ്പെട്ട പ്രോംപ്റ്റുകളും 90 ദിവസത്തേക്ക് മറയ്ക്കപ്പെടും.
  - ഒരു സ്റ്റാറ്റസ് സൂചന “Hidden until YYYY‑MM‑DD” (നിങ്ങളുടെ പ്രാദേശിക തീയതി) എന്നു കാണിക്കും. ഉടൻ ദൃശ്യമാക്കാൻ “Show Donate again” എന്ന ബട്ടണും ഉണ്ട്.
  - 90 ദിവസത്തിന് ശേഷം Donate ബട്ടൺ സ്വയമേവ വീണ്ടും ദൃശ്യമാകും.

- സ്വകാര്യതയും സംഭരണവും
  - സ്നൂസ് കാലയളവ് ഓർക്കാൻ Thunderbird‑ന്റെ ലൊക്കൽ സ്റ്റോറേജിൽ ഒരു ടൈംസ്റ്റാമ്പ് മാത്രം ആഡ്‑ഓൺ സൂക്ഷിക്കുന്നു. കീ: `donateHideUntil` (epoch milliseconds).
  - ഈ ക്രമീകരണം നിങ്ങളുടെ Thunderbird പ്രൊഫൈലിനോട് മാത്രം ബന്ധപ്പെട്ടതാണ് (ക്ലൗഡിലേക്ക് സിങ്ക് ചെയ്യുന്നില്ല). ഈ സവിശേഷത യാതൊരു നെറ്റ്‌വർക്ക് അഭ്യർത്ഥനയും ചെയ്യുന്നില്ല.

- പ്രശ്നപരിഹാരം
  - “I donated” ക്ലിക്ക് ചെയ്ത ഉടൻ പോലും Donate കാണിച്ചുതുടർന്നാൽ, കുറച്ച് നിമിഷം കാത്തിരിക്കുകയോ Options പേജ് വീണ്ടും തുറക്കുകയോ ചെയ്യുക; ക്രമീകരണം സേവ് ചെയ്ത ഉടൻ UI പുതുക്കപ്പെടും.
  - മാനുവലായി റീസെറ്റ് ചെയ്യാൻ “Show Donate again” ക്ലിക്ക് ചെയ്യുക. സൂചനയിൽ കാണിക്കുന്ന തീയതി കഴിയുന്നത് വരെ കാത്തിരിക്കാനും കഴിയും.

ഈ സവിശേഷത പൂർണ്ണമായും സൗകര്യത്തിനായി മാത്രമാണ്; ഇത് ഒരിക്കലും ആഡ്‑ഓൺ പ്രവർത്തനം തടസ്സപ്പെടുത്തുകയില്ല, വ്യക്തിഗത ഡാറ്റ ഒന്നും ശേഖരിക്കുകയുമില്ല.
