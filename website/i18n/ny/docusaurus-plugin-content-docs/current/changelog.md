## Mbiri ya zosintha

Kuti muone mbiri yonse yatsatanetsatane, onani
[CHANGELOG.md pa GitHub](https://github.com/bitranox/Thunderbird-Reply-with-Attachments/blob/master/CHANGELOG.md) ya repositori.

- 2.3.2: "Include inline pictures" tsopano imayika zithunzi mwachindunji m'thupi la yankho ngati base64 data URIs (default: ON); kusintha kwa khalidwe la code ndi test coverage yowonjezedwa.
- 2.3.1: Imasunga ma attachment ngakhale Thunderbird ikayika tsamba la zochitika lakumbuyo pa idle; imawonjezera ma debug hook olunjika pothandiza kuthetsa mavuto.
- 2.3.0: Kuwongolera bwino kuchotsa zobwerezabwereza za ma attachment, kukulitsa test coverage, komanso kuchotsa zilolezo zakale kuti zikwaniritse malamulo a AMO.
- 2.1.0: Thandizo lonse la internationalization pa zilankhulo 100 zapamwamba
- 2.0.0: kulemba kachiwiri kukhala mtundu wokwanira ndi mbali zonse (EN/DE)
- 1.0.1: yasinthira ku messages.listAttachments()
- 1.0.0: kutulutsa koyamba
