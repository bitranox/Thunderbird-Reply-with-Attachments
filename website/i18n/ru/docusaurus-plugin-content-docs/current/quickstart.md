## Быстрый старт

:::important Минимальная версия Thunderbird
Это дополнение поддерживает Thunderbird **128 ESR и новее**. Более старые версии не поддерживаются.
:::

:::note Без телеметрии; без фоновых сетевых запросов
Дополнение **не** собирает аналитику/телеметрию и **не** выполняет фоновых сетевых запросов. Доступ к сети происходит только при нажатии внешних ссылок (Документация, GitHub, Пожертвовать).
:::

---

### Установка

1. Установите дополнение с сайта Thunderbird Add‑ons.
2. Необязательно: включите подтверждение (Параметры → «Спрашивать перед добавлением вложений»).
3. Необязательно: оставьте включённым предупреждение о чёрном списке (по умолчанию): «Предупреждать, если вложения исключены чёрным списком».
4. Необязательно: добавьте шаблоны чёрного списка (по одному в строке), например:

```
*intern*
*secret*
*passwor*  # matches both “password” and “Passwort” families
```

Примечание: «# …» выше — это комментарий в этой документации; не включайте комментарии в шаблоны, которые вы вставляете в Параметры. Вводите только по одному шаблону на строку.

Теперь ответьте на сообщение с вложениями — исходные файлы будут добавлены автоматически или после быстрого подтверждения. Если какие‑либо файлы исключены вашим чёрным списком, вы увидите короткое предупреждение с их перечнем.

---

### Проверка {#verify}

- Ответьте на сообщение с 1–2 вложениями и убедитесь, что исходные файлы добавлены в окно создания письма.
- Чтобы настроить поведение, см. [Настройки](configuration) (переключатель подтверждения, ответ по умолчанию, шаблоны чёрного списка).

---

### Проверка предупреждения чёрного списка {#verify-blacklist-warning}

- Ответьте на сообщение, содержащее файл вроде «secret.txt».
- При включённой опции «Предупреждать, если вложения исключены чёрным списком» небольшое окно покажет список исключённых файлов и соответствующий шаблон.

Если предупреждение не появляется, убедитесь, что шаблон точно совпадает с именем файла (только имя файла, без учёта регистра). См. Настройки → Чёрный список.

---

### Примечание по клавиатуре {#keyboard-note}

- В окне подтверждения работают клавиши Y/J для Да и N/Esc для Нет. На некоторых нелатинских раскладках буквы могут отличаться; Enter подтверждает выделенную кнопку.
