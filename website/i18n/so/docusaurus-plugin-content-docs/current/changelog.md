## Diiwaanka isbeddellada

Si aad u aragto taariikhda dhamaystiran ee faahfaahsan, eeg [CHANGELOG.md ee GitHub](https://github.com/bitranox/Thunderbird-Reply-with-Attachments/blob/master/CHANGELOG.md).

- 2.3.2: "Ku dar sawirro ku dhex jira" hadda waxay si toos ah ugu dhex gelisaa sawirrada jidhka jawaabta iyadoo la adeegsanayo URI-yada xogta base64 (caadiga: ON); horumarinta tayada koodhka iyo ballaarinta daboolka tijaabooyinka.
- 2.3.1: Waxay sii haysaa lifaaqyada xitaa kadib marka Thunderbird ka dhigo bogga dhacdooyinka dambe mid aan firfircoonayn; waxay ku dartaa debug hooks bartilmaameedsan oo lagu fududeynayo cilad-raadin.
- 2.3.0: Waxaa la hagaajiyay ka-saaridda soo noqnoqoshada lifaaqyada, la ballaariyay daboolka tijaabooyinka, waxaana laga saaray oggolaanshiyo gaboobay si loo waafajiyo siyaasadaha AMO.
- 2.1.0: Taageero buuxda oo caalamiyeynta (i18n) ah oo loogu talagalay 100-ka luqadood ee ugu sarreeya
- 2.0.0: Dib-u-qoris nooc dhammeystiran (EN/DE)
- 1.0.1: Waxaa loo wareegay messages.listAttachments()
- 1.0.0: Sii-deyntii ugu horreysay
