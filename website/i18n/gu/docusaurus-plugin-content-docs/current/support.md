મદદ જોઈએ છે કે બગ રિપોર્ટ કરવો છે?

- GitHub પર એક issue ખોલો:
  - રિપોઝિટરી: `bitranox/Thunderbird-Reply-with-Attachments`
  - Thunderbird વર્ઝન (ઉદા. 128 ESR), OS, અને પુનઃઉત્પાદન પગલાં લખો
  - Thunderbird ના Error Console માંથી સંબંધિત લોગ્સ જોડો (Tools → Developer Tools → Error Console)

- Add‑ons સાઇટ (ATN): તમે ઍડ‑ઓન પેજ દ્વારા પ્રતિસાદ પણ આપી શકો છો.

સૂચનો

- ખાતરી કરો કે તમે સમર્થિત Thunderbird વર્ઝન (128 ESR અથવા નવું) વાપરી રહ્યા છો.
- સામાન્ય સેટઅપ પ્રશ્નો માટે Configuration અને Usage દસ્તાવેજો જુઓ.
- વિકાસ/ટેસ્ટિંગ માટે [Development](development) માર્ગદર્શિકા જુઓ.

જો તમે આ પ્રોજેક્ટને ટેકો આપવા માંગતા હોવ તો કૃપા કરીને [Donate](donation) પેજ પર નાનો ફાળો આપવાનો વિચાર કરો. આભાર!
