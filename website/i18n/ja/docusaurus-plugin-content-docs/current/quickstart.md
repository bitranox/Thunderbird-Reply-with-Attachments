## クイックスタート

:::important Thunderbird の最小対応バージョン
このアドオンは Thunderbird **128 ESR 以降**に対応しています。旧バージョンはサポートされません。
:::

:::note テレメトリなし・バックグラウンド通信なし
このアドオンは分析/テレメトリを収集することは**ありません**。また、バックグラウンドでネットワーク要求を行うことも**ありません**。ネットワークアクセスは、外部リンク（Docs、GitHub、Donate）をクリックしたときにのみ発生します。
:::

---

### インストール

1. Thunderbird Add‑ons からアドオンをインストールします。
2. 任意: 確認を有効にします（Options → “Ask before adding attachments”）。
3. 任意: ブラックリスト警告を有効のままにしておきます（既定）: “Warn if attachments are excluded by blacklist”。
4. 任意: ブラックリストのパターンを追加します（1 行に 1 つ）。例:

```
*intern*
*secret*
*passwor*  # matches both “password” and “Passwort” families
```

注: 上記の “# …” はこのドキュメント内のコメントです。Options に貼り付けるパターンにはコメントを含めないでください。1 行につき 1 つのパターンのみを入力してください。

次に、添付ファイルのあるメッセージに返信してみてください。元の添付は自動的に、または簡単な確認の後に追加されます。ブラックリストにより除外されたファイルがある場合は、それらの一覧を示す短い警告が表示されます。

---

### 検証 {#verify}

- 1～2 個の添付ファイルがあるメッセージに返信し、元の添付が作成ウィンドウに追加されることを確認します。
- 動作を調整するには、[設定](configuration) を参照してください（確認の切り替え、既定の回答、ブラックリストのパターン）。

---

### ブラックリスト警告の確認 {#verify-blacklist-warning}

- “secret.txt” のようなファイルを含むメッセージに返信します。
- “Warn if attachments are excluded by blacklist” を有効にしている場合、小さなダイアログに除外されたファイルと一致したパターンが一覧表示されます。

警告が表示されない場合は、パターンがファイル名に正確に一致していることを確認してください（ファイル名のみ・大文字小文字は区別しません）。設定 → Blacklist を参照してください。

---

### キーボードに関する注意 {#keyboard-note}

- 確認ダイアログでは、Yes は Y/J、No は N/Esc で操作できます。非ラテン系のキーボードでは文字キーが異なる場合があります。Enter でフォーカスされているボタンを確定します。
