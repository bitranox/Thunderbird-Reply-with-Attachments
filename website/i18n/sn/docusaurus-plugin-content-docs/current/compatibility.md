## Kuwirirana {#compatibility}

:::important Shanduro shoma yeThunderbird
Kuwedzerwa uku kunotsigira Thunderbird **128 ESR kana nyowani kupfuura**. Shanduro dzekare hadzitsigirwe.
:::

- Thunderbird 128 ESR kana nyowani kupfuura inotsigirirwa (128.x+).
- Shanduro dzekare dzeThunderbird hadzitsigirwe.
- Yakayedzwa paWindows, macOS, neLinux.

:::warning Manifest V2 haina kutsigirirwa
Manifest V2 haina kutsigirirwa. Kuwedzerwa kunonangana ne **Manifest V3 (MV3)** paThunderbird 128 ESR+.
:::

---

## Kugadzirisa Matambudziko {#troubleshooting}

- Kana kuwedzerwa kusingashande sezvatarisirwa, ita shuwa kuti uri kushandisa shanduro inowirirana yeThunderbird (128 ESR kana kupfuura).
- Tarisa koni yemakanganiso yeThunderbird (**Tools > Developer Tools > Error Console**) kune chero nyaya dzine chekuita nekuwedzerwa.
- Kana marongero akachengetwa achiratidza kusashanda nemazvo, tangazve Thunderbird wozoyedza zvakare. (Thunderbird inogona kuchengetedza mamiriro pakati pemisesheni; kutangazve kunoita shuwa kuti marongero matsva akarodha.)

## Kusawirirana Kunozivikanwa {#known-conflicts}

- Hapana anozivikanwa pari zvino. Kana ukaona zvinobatanidzwa zvinodzokororwa kaviri kana kukundikana nemamwe ekuwedzerwa, tapota nyora mushumo uchiisa nhanho dzekudzokorora.

---

## Beta/Nightly {#beta-nightly}

- Shanduro dzeThunderbird Beta neDaily (Nightly) dzinowanzotsigirirwa, asi shanduko dzinobva kumusoro dzimwe nguva dzinogona kuputsa maitiro ekuwedzerwa. Kana ukasangana nematambudziko, tapota azivise uye ubatanidze shanduro chaiyo yeThunderbird (semuenzaniso, “Daily 131.0a1 2025‑09‑10”).
- Wasangana nekudzokera shure paBeta/Nightly? Tapota zivisa uine nhanho dzekudzokorora — ona [Rubatsiro](support) — uye sanganisira build string chaiyo yeThunderbird.
