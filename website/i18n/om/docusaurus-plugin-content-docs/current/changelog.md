## Gabaasa Jijjiiramaa

Seenaa guutuu fi bal'aaaf, kan reposiitarii [CHANGELOG.md GitHub irratti](https://github.com/bitranox/Thunderbird-Reply-with-Attachments/blob/master/CHANGELOG.md) ilaali.

- 2.3.2: "Include inline pictures" amma suuraalee qaama deebii keessaatti kallattiidhaan akka base64 data URIsitti maxxansiisa (haal-durtii: ON); qulqullina koodii fooyya'e fi bal'inni gahee qorannoo bal'ate.
- 2.3.1: Thunderbird fuula taatee duubaa idle gochuu booda illee, faayiloota itti maxxanfaman ni eega; rakkoo hiikuuf 'debug hooks' xiyyeeffatan dabalata.
- 2.3.0: Duplikeeshinii faayiloota itti maxxanfaman fooyyeesse, gahee qorannoo bal'ise, akkasumas imaammata AMO guutuuf hayyamoota hojii ala ta'an haqe.
- 2.1.0: Deeggarsa guutuu internaashinaalaayizeeshinii (i18n) afaanota 100 isa duraa'f
- 2.0.0: Irra deebi'ee gara versiinii amala-guutuu qabuutti barreesse (EN/DE)
- 1.0.1: gara messages.listAttachments()tti jijjiire
- 1.0.0: maxxansa jalqabaa
