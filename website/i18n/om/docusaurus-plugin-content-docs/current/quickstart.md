## Jalqabsi saffisaa

:::important Vershinii Thunderbird Minimaa
Add‑on kun Thunderbird **128 ESR ykn isa haaraa** ni deeggara. Vershinoonni duraanii hin deeggaraman.
:::

:::note Telemeeterii hin jiru; network duubaa hin jiru
Add‑on kun qo’annoo/telemeeterii **hin** sassaabu, akkasumas gaaffii network duubaa **hin** godha. Seensa network linkoota alaa (Docs, GitHub, Donate) tuqtanitti qofa ni ta’a.
:::

---

### Install

1. Add-on kana Thunderbird Add‑ons irraa install godhaa.
2. Filannoo: Mirkaneessa dandeessisi (Options → “Ask before adding attachments”).
3. Filannoo: Akeekkachiisa blacklist dandeessifame (durtii) akka hafu dhiisi: “Warn if attachments are excluded by blacklist”.
4. Filannoo: Paatarnoota blacklist dabaluu (sarara tokkoo irratti tokkicha), fakkeenyaaf:

```
*intern*
*secret*
*passwor*  # matches both “password” and “Passwort” families
```

Yaadachiisa: “# …” armaan olii kun yaada (comment) qajeelfama kana keessatti; paatarnoota Options keessa maxxanfattu keessatti yaadoota hin dabalinaa. Paatarnii tokkichi sarara tokkicha qofa galchaa.

Amma ergaa faayilota itti makaman waliin deebisi — asiliin ofumaan yookaan mirkaneessa saffisaa booda dabalama. Faayiloonni tokko tokkos blacklist keessan irraan dhoorkaman yoo jiraatan, isaan tarreessu akeekkachiisa gabaabaa ni argita.

---

### Mirkaneessi {#verify}

- Ergaa makaa 1–2 qabu deebisi; originals foddaa barreessuu keessaniitti dabalaman akka ta’e mirkaneessi.
- Akkaataa hojii isaa fooyyessuuf, [Qindaa’ina](configuration) ilaali (toggle mirkaneessaa, deebii durtii, paatarnoota blacklist).

---

### Akeekkachiisa blacklist mirkaneessi {#verify-blacklist-warning}

- Faayila akkuma “secret.txt” of keessaa qabu ergaa deebisi.
- “Warn if attachments are excluded by blacklist” dandeessifamee yoo jiraate, daayalogi xiqqaan faayilota dhoorkaman fi paatarnii walsimu tarreessa.

Yoo akeekkachiisa hin argine, paatarniin maqaa faayilaatti guutumaan guutuutti akka walsimu mirkaneessi (maqaa faayilaa qofa, adda addummaan qubee hin ilaalamu). Qindaa’ina → Blacklist ilaali.

---

### Yaadannoo Kiiboordii {#keyboard-note}

- Daayalogi mirkaneessaa Y/J Eeyyeef fi N/Esc Lakkif ni deeggara. Kiiboordiiwwan muraasa warra Laatiin hin taane irratti furtuuleen qubee ni garaagaru; Enter bantiin irratti xiyyeeffate ni mirkaneessa.
