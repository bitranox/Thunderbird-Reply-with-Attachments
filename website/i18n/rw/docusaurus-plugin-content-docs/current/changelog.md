## Impinduka

Kugira ngo ubone amateka arambuye yose, reba [CHANGELOG.md kuri GitHub](https://github.com/bitranox/Thunderbird-Reply-with-Attachments/blob/master/CHANGELOG.md) y’ububiko.

- 2.3.2: "Gushyiramo amafoto mu butumwa" ubu ishyira amafoto mu mubiri w’igisubizo nk’amasano (URI) ya data ya base64 (mburabuzi: ON); kunoza ubuziranenge bwa kode no kwagura aho igeragezwa rigerera.
- 2.3.1: Ikomeza inyongeracyandiko (attachments) nyuma y’uko Thunderbird ishyize background event page mu gihe cyo kuruhuka (idle); yongeraho targeted debug hooks zo gufasha gukemura ibibazo.
- 2.3.0: Yanonosoye gukuraho kopi zisubiramo ku nyongeracyandiko (deduplication), yagaguye aho igeragezwa rigerera, kandi yakuyemo uburenganzira butagikenewe kugira ngo huzuzwe amabwiriza ya AMO.
- 2.1.0: Ubufasha bwuzuye bwo guhinduranya indimi (internationalization) ku ndimi 100 za mbere
- 2.0.0: Kwandikwa bushya bukaba verisiyo ifite ibiranga byose (EN/DE)
- 1.0.1: Yimukiye kuri messages.listAttachments()
- 1.0.0: Isosora rya mbere
