## Tala ng mga Pagbabago

Para sa kumpleto at detalyadong kasaysayan, tingnan ang
[CHANGELOG.md sa GitHub](https://github.com/bitranox/Thunderbird-Reply-with-Attachments/blob/master/CHANGELOG.md).

- 2.3.2: Ang "Isama ang mga larawang inline" ngayon ay direktang nag-e-embed ng mga larawan sa katawan ng tugon bilang mga base64 data URI (default: ON); mga pagpapahusay sa kalidad ng code at pinalawak na saklaw ng mga pagsubok.
- 2.3.1: Pinananatili ang mga attachment kapag ini-idle ng Thunderbird ang background event page; nagdaragdag ng mga nakatutok na debug hooks para sa pag-troubleshoot.
- 2.3.0: Pinahusay ang deduplication ng attachment, pinalawak ang saklaw ng mga pagsubok, at inalis ang mga lipas na permission upang matugunan ang mga patakaran ng AMO.
- 2.1.0: Ganap na suporta sa internasyunalisasyon para sa nangungunang 100 wika
- 2.0.0: muling isinulat tungo sa isang ganap na tampok na bersyon (EN/DE)
- 1.0.1: lumipat sa messages.listAttachments()
- 1.0.0: unang paglabas

## Mga Petsa at Channel {#dates-and-channels}

- Maaaring maantala ng ilang oras ang mga release sa ATN pagkatapos ng pag-package.
- Ang mga LOCAL build ay para lamang sa pagsubok ng mga developer at hindi ipinapamahagi sa pamamagitan ng ATN.
