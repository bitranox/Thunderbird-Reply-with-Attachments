uradni seznam v drevesu izvorne kode glejte `THIRD_PARTY_LICENSES.md` v
korenini repozitorija.

Opombe

- Thunderbirdovi API-ji MailExtension so platformni API-ji; iz njih ni vključene nobene kode tretjih oseb.
- Ikone projekta v `sources/icons` so projektna sredstva (MIT, razen če je navedeno drugače). Logotip/ikona GitHub je blagovna znamka GitHub in ni zajeta v MIT; uporablja se skladno s smernicami blagovne znamke GitHub.

Če dodate nove odvisnosti ali vključite kodo tretjih oseb, ustrezno posodobite tako to
stran kot tudi `THIRD_PARTY_LICENSES.md`.
