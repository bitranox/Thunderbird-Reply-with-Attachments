## Catetan Parobahan

Pikeun sajarah anu lengkep jeung rinci, tingali repositori
[CHANGELOG.md di GitHub](https://github.com/bitranox/Thunderbird-Reply-with-Attachments/blob/master/CHANGELOG.md).

- 2.3.2: "Include inline pictures" ayeuna nyelapkeun gambar langsung kana awak balesan salaku data URI base64 (bawaan: ON); paningkatan kualitas kode jeung ngalegaan cakupan tés.
- 2.3.1: Nahan lampiran sanggeus Thunderbird ngajadikeun kaca kajadian latar tukang jadi idle; nambahkeun kait debug anu husus dituju pikeun ngungkulan masalah.
- 2.3.0: Dibeuki hadé deduplikasi lampiran, ngalegaan cakupan tés, sarta miceun idin anu geus teu diperlukeun deui pikeun nyumponan kawijakan AMO.
- 2.1.0: Rojongan internasionalisasi lengkep pikeun 100 basa pangluhurna
- 2.0.0: Ditulis deui jadi vérsi pinuh-fitur (EN/DE)
- 1.0.1: Diganti kana messages.listAttachments()
- 1.0.0: Rilis awal
