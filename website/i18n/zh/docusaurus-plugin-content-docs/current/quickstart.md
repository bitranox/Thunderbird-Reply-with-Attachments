## 快速开始

:::important 最低 Thunderbird 版本
此附加组件支持 Thunderbird **128 ESR 或更新版本**。不支持更早的版本。
:::

:::note 无遥测；无后台网络请求
该附加组件**不会**收集分析/遥测数据，且**不会**发起任何后台网络请求。只有当你点击外部链接（文档、GitHub、捐赠）时才会进行网络访问。
:::

---

### 安装

1. 从 Thunderbird Add‑ons 安装此附加组件。
2. 可选：启用确认（选项 → “添加附件前询问”）。
3. 可选：保持启用黑名单警告（默认）： “当附件被黑名单排除时发出警告”。
4. 可选：添加黑名单模式（每行一个），例如：

```
*intern*
*secret*
*passwor*  # matches both “password” and “Passwort” families
```

注意：上述 “# …” 在本文档中表示注释；在粘贴到“选项”中的模式里不要包含注释。每行只输入一个模式。

现在回复一封带有附件的邮件——原始附件将自动添加，或在快速确认后添加。若有文件被你的黑名单排除，你会看到一个简短的警告，列出这些文件。

---

### 验证 {#verify}

- 回复一封包含 1–2 个附件的邮件，并确认原始附件已添加到你的撰写窗口。
- 如需调整行为，请参见[配置](configuration)（确认开关、默认回答、黑名单模式）。

---

### 验证黑名单警告 {#verify-blacklist-warning}

- 回复一封包含类似 “secret.txt” 文件的邮件。
- 启用“当附件被黑名单排除时发出警告”后，会弹出一个小对话框，列出被排除的文件及其匹配的模式。

如果你没有看到警告，请确保该模式与文件名完全匹配（仅针对文件名，大小写不敏感）。参见 配置 → 黑名单。

---

### 键盘说明 {#keyboard-note}

- 确认对话框支持使用 Y/J 表示“是”，N/Esc 表示“否”。在某些非拉丁键盘上，字母键可能不同；Enter 将确认当前聚焦的按钮。
