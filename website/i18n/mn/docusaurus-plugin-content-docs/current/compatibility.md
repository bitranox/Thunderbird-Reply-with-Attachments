## Тохиромжтой байдал {#compatibility}

:::important Thunderbird-ийн хамгийн бага хувилбар
Энэ нэмэлт нь Thunderbird **128 ESR эсвэл түүнээс шинэ** хувилбаруудыг дэмжинэ. Хуучин хувилбарууд дэмжигдэхгүй.
:::

- Thunderbird 128 ESR эсвэл түүнээс шинэ нь дэмжигдэнэ (128.x+).
- Thunderbird‑ийн хуучин хувилбарууд дэмжигдэхгүй.
- Windows, macOS, Linux дээр туршсан.

:::warning Manifest V2 дэмжигдэхгүй
Manifest V2 дэмжигдэхгүй. Нэмэлт нь Thunderbird 128 ESR+ дээр **Manifest V3 (MV3)**‑г онилсон.
:::

## Алдаа засварлах {#troubleshooting}

- Нэмэлт хүлээж буйгаар ажиллахгүй бол Thunderbird‑ийн тохирох хувилбар (128 ESR эсвэл түүнээс хойшх) ашиглаж буйгаа шалгаарай.
- Thunderbird‑ийн алдааны консолд (**Tools > Developer Tools > Error Console**) нэмэлттэй холбоотой асуудал байгаа эсэхийг шалгаарай.
- Хадгалсан тохиргоо зөв хэрэгжихгүй байгаа мэт санагдвал Thunderbird‑ийг дахин асааж туршаад үзнэ үү. (Thunderbird нь төлөвийг сессүүдийн хооронд кэшлэж болдог; дахин асаалт нь шинэ тохиргоог цэвэр байдлаар ачаалагдахыг баталгаажуулна.)

## Мэдэгдэж буй зөрчилдөөнүүд {#known-conflicts}

- Одоогоор мэдэгдсэн зүйлгүй. Хэрэв давтагдсан, давхардсан хавсралтууд эсвэл бусад нэмэлтүүдтэй хамт алдаа давтан гарч байвал, дахин гаргах алхмуудтай нь хамт мэдээллээрэй.

---

## Бета/Найтли {#beta-nightly}

- Thunderbird‑ийн Beta болон Daily (Nightly) барилтууд ихэвчлэн дэмжигддэг, гэхдээ суурь өөрчлөлтүүд заримдаа нэмэлтийн үйлдлийг эвдэж болзошгүй. Асуудал таарвал заавал мэдээлж, Thunderbird‑ийн яг тийм хувилбарыг оруулна уу (жиш., “Daily 131.0a1 2025‑09‑10”).
- Бета/Найтли дээр регресс таарсан уу? Дахин гаргах алхмуудтай нь хамт — [Дэмжлэг](support) — хуудас руу орж мэдээлээд, Thunderbird‑ийн барилтын яг тийм мөрийг оруулна уу.
