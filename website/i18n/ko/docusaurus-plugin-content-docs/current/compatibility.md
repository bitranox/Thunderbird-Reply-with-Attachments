## 호환성 {#compatibility}

:::important 최소 Thunderbird 버전
이 부가 기능은 Thunderbird **128 ESR 이상**을 지원합니다. 이전 버전은 지원되지 않습니다.
:::

- Thunderbird 128 ESR 이상을 지원합니다(128.x+).
- 이전 Thunderbird 버전은 지원되지 않습니다.
- Windows, macOS, Linux에서 테스트되었습니다.

:::warning Manifest V2는 지원되지 않음
Manifest V2는 지원되지 않습니다. 이 부가 기능은 Thunderbird 128 ESR+에서 **Manifest V3 (MV3)**를 대상으로 합니다.
:::

---

## 문제 해결 {#troubleshooting}

- 부가 기능이 예상대로 작동하지 않는 경우, 호환되는 Thunderbird 버전(128 ESR 이상)을 사용 중인지 확인하세요.
- 부가 기능 관련 문제는 Thunderbird 오류 콘솔(**Tools > Developer Tools > Error Console**)에서 확인하세요.
- 저장된 설정이 제대로 적용되지 않는 것 같으면 Thunderbird를 다시 시작한 다음 다시 시도하세요. (Thunderbird는 세션 간 상태를 캐시할 수 있습니다. 다시 시작하면 새 설정이 로드됩니다.)

## 알려진 충돌 {#known-conflicts}

- 현재 알려진 문제가 없습니다. 중복 첨부가 반복되거나 다른 부가 기능과 충돌/실패가 발생하면, 재현 절차와 함께 보고해 주세요.

---

## 베타/나이트리 {#beta-nightly}

- Thunderbird Beta 및 Daily(Nightly) 빌드는 일반적으로 지원되지만, 상류 변경으로 인해 때때로 부가 기능 동작이 깨질 수 있습니다. 문제가 발생하면 정확한 Thunderbird 버전(예: “Daily 131.0a1 2025‑09‑10”)을 포함해 보고해 주세요.
- Beta/Nightly에서 회귀 문제가 발생했나요? 재현 절차와 함께 보고해 주세요 — [Support](support)를 참조 — 그리고 정확한 Thunderbird 빌드 문자열을 포함해 주세요.
