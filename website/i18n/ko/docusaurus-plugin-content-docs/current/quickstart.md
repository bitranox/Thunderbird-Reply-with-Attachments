## 빠른 시작

:::important 최소 Thunderbird 버전
이 애드온은 Thunderbird **128 ESR 이상**을 지원합니다. 더 낮은 버전은 지원되지 않습니다.
:::

:::note 텔레메트리 없음; 백그라운드 네트워크 없음
이 애드온은 분석/텔레메트리를 수집하지 **않으며**, 백그라운드 네트워크 요청을 **하지 않습니다**. 네트워크 액세스는 외부 링크(문서, GitHub, 기부)를 클릭할 때만 발생합니다.
:::

---

### 설치

1. Thunderbird 애드온에서 애드온을 설치하세요.
2. 선택 사항: 확인 기능을 활성화하세요(옵션 → “첨부 파일 추가 전에 묻기”).
3. 선택 사항: 블랙리스트 경고를 활성화 상태로 유지하세요(기본값): “블랙리스트로 인해 첨부 파일이 제외되면 경고”.
4. 선택 사항: 블랙리스트 패턴을 추가하세요(한 줄에 하나씩). 예:

```
*intern*
*secret*
*passwor*  # matches both “password” and “Passwort” families
```

참고: 위의 “# …”는 이 문서에서의 주석 예시입니다. 옵션에 패턴을 붙여 넣을 때 주석은 포함하지 마세요. 한 줄에 하나의 패턴만 입력하세요.

이제 첨부 파일이 있는 메시지에 답장해 보세요 — 원본이 자동으로 추가되거나 간단한 확인 후 추가됩니다. 블랙리스트에 의해 어떤 파일이 제외되면, 해당 파일을 나열하는 짧은 경고가 표시됩니다.

---

### 확인 {#verify}

- 첨부 파일이 1–2개 있는 메시지에 답장하고, 원본이 작성 창에 추가되었는지 확인하세요.
- 동작을 조정하려면 [구성](configuration)을 참고하세요(확인 토글, 기본 답변, 블랙리스트 패턴).

---

### 블랙리스트 경고 확인 {#verify-blacklist-warning}

- “secret.txt” 같은 파일이 포함된 메시지에 답장하세요.
- “블랙리스트로 인해 첨부 파일이 제외되면 경고”가 활성화된 경우, 작은 대화상자에 제외된 파일과 일치하는 패턴이 표시됩니다.

경고가 보이지 않는다면, 패턴이 파일명과 정확히 일치하는지 확인하세요(파일명만, 대소문자 구분 안 함). 구성 → 블랙리스트를 참고하세요.

---

### 키보드 참고 {#keyboard-note}

- 확인 대화상자에서는 예(Yes)는 Y/J, 아니오(No)는 N/Esc 키를 지원합니다. 일부 비라틴 키보드에서는 글자 키가 다를 수 있습니다. Enter 키를 누르면 포커스된 버튼이 확인됩니다.
