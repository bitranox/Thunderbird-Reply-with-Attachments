## အမြန်စတင်

:::important အနည်းဆုံး Thunderbird မူကွဲ
ဤ add‑on သည် Thunderbird **128 ESR သို့မဟုတ် ထို့ထက် အသစ်များ** ကိုပံ့ပိုးပါသည်။ အဟောင်း မူကွဲများကို မပံ့ပိုးပါ။
:::

:::note Telemetry မရှိ၊ နောက်ခံကွန်ယက် မရှိ
ဤ add‑on သည် analytics/telemetry ကို **မ**စုဆောင်းသလို နောက်ခံ ကွန်ယက်တောင်းဆိုမှုများကိုလည်း **မ**ပြုလုပ်ပါ။ ကွန်ယက်သုံးစွဲမှုသည် သင် ပြင်ပလင့်ခ်များ (Docs, GitHub, Donate) ကို နှိပ်သည့်အချိန်တွင်သာ ဖြစ်ပေါ်ပါသည်။
:::

---

### တပ်ဆင်ခြင်း

1. Thunderbird Add‑ons မှ add‑on ကို တပ်ဆင်ပါ။
2. ရွေးချယ်နိုင်သည်: အတည်ပြုမှုကို ဖွင့်ပါ (Options → “Ask before adding attachments”).
3. ရွေးချယ်နိုင်သည်: blacklist သတိပေးချက်ကို ဖွင့်ထားပါ (default): “Warn if attachments are excluded by blacklist”.
4. ရွေးချယ်နိုင်သည်: blacklist ပုံစံများကို ထည့်ပါ (တစ်ကြောင်းလျှင် တစ်ပုံစံ), ဥပမာ:

```
*intern*
*secret*
*passwor*  # matches both “password” and “Passwort” families
```

မှတ်ချက်: အထက်ရှိ “# …” သည် ဤစာရွက်စာတမ်းအတွက် မှတ်ချက် ဖြစ်ပါသည်; Options ထဲသို့ ပုံစံများကို ကူးထည့်ရာတွင် မှတ်ချက်များ မထည့်ပါနှင့်။ တစ်ကြောင်းလျှင် ပုံစံ တစ်ခုသာ ထည့်ပါ။

ယခု တွဲဖိုင်များ ပါဝင်သည့် မက်ဆေ့ချ်ကို ပြန်စာပို့ပါ — မူရင်း တွဲဖိုင်များကို အလိုအလျှောက် သို့မဟုတ် အတည်ပြုချက်တိုတစ်ခုပြီးနောက် ထည့်သွင်းပေးမည် ဖြစ်သည်။ သင့် blacklist ကြောင့် ဖယ်ထုတ်ထားသော ဖိုင်များရှိပါက ၎င်းများကို စာရင်းပြပေးသော သတိပေးချက်တို တစ်ခုပေါ်လာပါမည်။

---

### စစ်ဆေးခြင်း {#verify}

- တွဲဖိုင် 1–2 ခု ပါဝင်သည့် မက်ဆေ့ချ်ကို ပြန်စာပို့ပြီး မူရင်း တွဲဖိုင်များကို သင့် ရေးသားခြင်း ပြတင်းပေါက်တွင် ထည့်သွင်းထားကြောင်း အတည်ပြုပါ။
- လုပ်ဆောင်မှုကို ချိန်ညှိရန် [Configuration](configuration) ကို ကြည့်ပါ (အတည်ပြုမှု ပြောင်းဖွင့်/ပိတ်, ပုံသေ အဖြေ, blacklist ပုံစံများ)။

---

### တားမြစ်စာရင်း သတိပေးချက် စစ်ဆေးခြင်း {#verify-blacklist-warning}

- “secret.txt” ကဲ့သို့ ဖိုင် ပါရှိသော မက်ဆေ့ချ်ကို ပြန်စာပို့ပါ။
- “Warn if attachments are excluded by blacklist” ကို ဖွင့်ထားလျှင် ဖယ်ထုတ်ထားသော ဖိုင်များနှင့် ကိုက်ညီသည့် ပုံစံကို စာရင်းပြသသော ဒီယာလော့ဂ်ငယ်တစ်ခု ပေါ်လာပါမည်။

သတိပေးချက် မမြင်ရပါက ပုံစံသည် ဖိုင်နာမည်နှင့် တိတိကျကျ ကိုက်ညီနေလား (ဖိုင်နာမည်ပင်သာ၊ စာလုံးအကြီးအသေး မခွဲခြား) ကို စစ်ဆေးပါ။ Configuration → Blacklist ကို ကြည့်ပါ။

---

### ကီးဘုတ် မှတ်ချက် {#keyboard-note}

- အတည်ပြုချက် dialog တွင် Yes အတွက် Y/J နှင့် No အတွက် N/Esc ကို အသုံးပြုနိုင်သည်။ Latin မဟုတ်သည့် ကီးဘုတ်အချို့တွင် အက္ခရာခလုတ်များ ကွဲပြားနိုင်သည်; Enter သည် အာရုံစိုက်ထားသော ခလုတ်ကို အတည်ပြုသည်။
