မှတ်ချက်များ

- လှူဒါန်းရန် လင့်ခ်များသည် သင်နှိပ်သော်လည်းမှ ဖွင့်မည်ဖြစ်ပြီး၊ အက်ဒ်အွန်သည် နောက်ခံကွန်ယက် တောင်းဆိုမှုများကို မလုပ်ဆောင်ပါ။
- အကြိမ်ကြိမ် ပံ့ပိုးမှုသည် ရေရှည် ပြုပြင်ထိန်းသိမ်းမှုနှင့် အချက်အလက် အပ်ဒိတ်များကို သင့်တော်ချိန်တိုင်း ပေးအပ်နိုင်ရန် အထောက်အကူပြုသော်လည်း၊ လုံးဝ စိတ်ကြိုက် ဖြစ်ပါသည်။

## လှူဒါန်းမှု ပြသမှု (ရက် ၉၀ ခဏဖျောက်)

ဤအက်ဒ်အွန်တွင် သင် လှူဒါန်းပြီးနောက် ကာလတစ်ခုအတွင်း လှူဒါန်းရန် တိုက်တွန်းချက်များကို ခဏ ဖျောက်ထားနိုင်ရန် အဆင်ပြေမှုအင်္ဂါရပ်တစ်ခု ပါဝင်သည်။

- ဘယ်မှာတွေ့ရမလဲ
  - Options → Support အပိုင်းတွင် “I donated” ခလုတ်နှင့် အကြံပြုချက် စာသားနေရာသေးစိတ်တစ်ခုကို တွေ့မြင်ရပါမည်။
  - Send‑confirmation dialog ထဲတွင်လည်း Donate ခလုတ်ကို ပြသထားသည်؛ snooze အခြေအနေ လုပ်ဆောင်နေစဥ် တွင် အလိုအလျောက် ဖျောက်သွားပါသည်။

- အလုပ်လုပ်ပုံ
  - “I donated” ကို နှိပ်လိုက်သောအခါ လှူဒါန်း ခလုတ်များနှင့် သက်ဆိုင်သော တိုက်တွန်းချက်များကို ရက် ၉၀ အတွက် ဖျောက်ထားပါသည်။
  - အခြေအနေ အကြံပြုချက်တွင် “Hidden until YYYY‑MM‑DD” (သင့်ဒေသ သတ်မှတ်ရက်စွဲဖြင့်) ဟုပြသပါသည်။ ချက်ချင်း ပြန်ပြသလိုပါက “Show Donate again” ခလုတ်ကို နှိပ်နိုင်ပါသည်။
  - ရက် ၉၀ ပြည့်သောအခါ Donate ခလုတ်သည် အလိုအလျောက် ပြန်မြင်ရပါမည်။

- ကိုယ်ရေးကိုယ်တာနှင့် သိမ်းဆည်းမှု
  - snooze ကာလကို မှတ်မိစေရန်အတွက် Thunderbird ၏ local storage တွင် timestamp တစ်ခုကိုသာ သိမ်းဆည်းပါသည်။ Key: `donateHideUntil` (epoch milliseconds)။
  - ယင်းသတ်မှတ်ချက်သည် သင်၏ Thunderbird profile နှင့် သီးသန့် (cloud-synced မဟုတ်) ဖြစ်သည်။ ဤအင်္ဂါရပ်အားဖြင့် ကွန်ယက်တောင်းဆိုမှုများကို မပြုလုပ်ပါ။

- ပြဿနာဖြေရှင်းခြင်း
  - “I donated” ကို နှိပ်ပြီးချက်ချင်း Donate သ هنوز ပြနေသေးပါက ခဏစောင့်ပါ သို့မဟုတ် Options စာမျက်နှာကို ပြန်ဖွင့်ပါ။ သတ်မှတ်ချက် သိမ်းဆည်းပြီးချင်း UI သည် ချက်ချင်း အပ်ဒိတ်ဖြစ်ပါမည်။
  - လက်ဖြင့် ပြန်အစပြုလိုပါက “Show Donate again” ကို နှိပ်ပါ။ သို့မဟုတ် အကြံပြုချက်တွင် ဖော်ပြထားသည့် ရက်စွဲ ရောက်သည်အထိ စောင့်နိုင်ပါသည်။

ဤအင်္ဂါရပ်သည် အဆင်ပြေစေရန်အတွက်သာ ဖြစ်ပြီး အက်ဒ်အွန်၏ လုပ်ဆောင်ချက်များကို မတားဆီးသကဲ့သို့ ကိုယ်ရေးကိုယ်တာ ဒေတာများကိုလည်း မစုဆောင်းပါ။
